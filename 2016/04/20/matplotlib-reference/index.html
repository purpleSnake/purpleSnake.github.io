Pyplot

pyplot是一系列命令式的函数，使得matplotlib以类似Matlab plot的形式工作。每个函数会对图片产生一定的作用。
一个简单的例子：

import matplotlib as plt
plt.plot([1,2,3,4])
plt.ylabel('some numbers')
plt.show()

和matlab很类似，不用学了

def f(t):
    return np.exp(-t)*np.cos(2*np.pi*t)

t1 = np.arange(0.0,5.0,0.1)
t2 = np.arange(0.0,5.0, 0.02)

plt.figure(1)
plt.subplot(211)
plt.plot(t1,f(t1),'bo', t2,f(t2),'k')

plt.subplot(212)
plt.plot(t2,np.cos(2*np.pi*t2),'r--')
plt.show()

Text

text box

很灵活。但是比较麻烦，不如matlab figure界面直接操作来的直接。
使用text()函数，接收bbox关键字参数，则可在文字周围画一个边框。

bbox_props = dict{boxstyle="rarrow,pad=0.3", fc="cyan", ec="b", lw=2}
t = ax.text(0,0,"Direction", ha="center",va="center", rotation=45, size=15, bbox=bbox_props)
bb = t.get_bbox_path()  #获取与文字相关联的对象
bb.set_boxstyle("rarrow",pad=0.6)

bbox style:circle,darrow(double arrow),larrow(left arrow),
rarrow(right arrow),round(圆角矩形),round4,roundtooth(波浪线矩形),
sawtooth(锯齿矩形),square.

text arrow

画一个连接两个点的箭头，附带说明文字。

ax.annotate("annotate", 
            xy=(x1,y1), xycoords='data', #箭头所在的点，对应的坐标系
            xytext=(x2,y2),textcoords='offset points' #文字所在的位置，坐标表示偏移量
            arrowprops = dict{arrowstyle="->", connectionstyle="arc3"}  #箭头属性
            )
