Array

Array是多维数组，numpy中最重要的概念，又称ndarray。numpy中，数组的维数称为axes，axes的个数称为rank。axes的长度代表数据长度。例如[1,2,3],rank为1，axis的长度为3。
Numpy.array与Python标准库中Array.array不同，Array.array是一维、长度可变列表，但提供更少的功能。ndarray中重要的属性有：

  • ndarray.ndim rank的值。
  • ndarray.shape tuple(元组)，(rows,columns)
  • ndarray.size 所有元素的个数
  • ndarray.itemsize 每个元素的占据字节的个数
  • ndarray.data 数据

Array creation

>>> import numpy as np
>>> np.array([1,2,3])

>>> np.zeros((3,4))

>>> np.ones((1,2))

>>> np.empty((2,3))

>>> np.arange(start,end,interval)
>>> np.arange(10,30,5)
array([10,15,20,25])

>>> np.linespace(start,end, dataNums)

>>> np.fromfunction  #从函数返回值构建array
>>> np.fromfile      #从文件构建array
>>> np.loadtxt          #

Basic operations

ndarray的操作都是elementwise的，将会产生一个新的array。
注意：*号同样也是elementwise的，矩阵的乘法使用np.dot.
一些类似+=,*=的操作符，将直接修改array，而不是新建。
array类内置了一些函数：np.sum(),np.min(),np.max()等等

Universal functions

sin,cos,exp等等
all,any,where等

Indexing,Slicing and Iterating

和array类似。

Shaping manipulation

reshape

>>> a = np.random.random((3,4))
>>> a.shape

>>> a.ravel()   #flatten the array
>>> a.shape = (6,2)

>>> a.resize((2,6))
>>> a.reshape((3,-1))  #-1的话，该维度自动计算

stacking together different arrays

将几个不同的数组组合成一个数组。

>>> np.vstack((a,b))  # 垂直方向堆叠
>>> np.hstack((a,b))  # 水平方向堆叠

Copies and view

>>> b = a  #no copy

>>> c = a.view()  # c is a view of the data owned by a. 类似于共享数据，但有各自的数据头。
                  # 可以有不同的维数。 修改数据会相互影响。
>>> c is a
False
>>> c.base is a
True     

>>> d = a.copy()  #deep copy

more fancy indexing

使用数组，bool数组对数组进行索引，很炫酷的操作。

Linear algebra

simple array operations

>>> a = np.array([[1,2],[3,4]])

>>> a.transpos()  #转置
>>> np.linalg.inv(a) #求逆
>>> np.dot(a,b)  #矩阵乘法
>>> np.trace(a)     #矩阵的迹
>>> np.linalg.sove(a,y) #求解线性方程组
>>> np.linalg.eig(a)   #矩阵特征值和特征向量
