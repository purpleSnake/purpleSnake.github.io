前言
Lsip是一种备受推崇的语言，如“独孤九剑”一样，学会了就能豁然开朗，给你一个全新的世界。
我本着“这么牛逼的东西，一定要去看看”的想法，来学习一下。万一顿悟点啥呢，万一能仗剑天涯了呢。
想法还是要有的，万一实现了呢。

学习资料：ANSI Common Lisp 中文翻译版

第一章 简介

Lisp很牛逼。学习Lisp不仅教你学会一种新的语言，它教你崭新的并且更强大的程序思考方法。
反正就是很牛逼。 学了你不吃亏，学了你不上当。
就这样。爱学不学。牛逼的名词现在讲了也不懂，还是不讲了。

第二章 欢迎来到Lisp

2.1 form

Lisp是一门交互语言。这意味着有个交互窗口，类似CMD窗口，输入命令，得到结果。
Lisp尝试执行所有，比如 尝试执行+

> (+ 2 3)
5

Lisp中一切皆是函数。

2.2 evaluation

Lisp尝试执行一切。
避免被执行，可以使用quote或者单引号'

> (quote (+ 3 5))
(+ 3 5)

2.3 data

Lisp 提供了所有在其他语言找得到，以及找不到的数据类型。
字符串，使用双引号包括的字符串，比如："ora et labora"。整数和字符串一样，都是对自身求值的。
有两个通常在别的语言找不到的Lisp数据类型是符号(symbol)与列表(lists)。
符号是英语的单词(word)。无论怎么输入，都会被转换为大写。符号通常不对自身求值，所以想引用符号，必须使用'引用它

> 'Benjamin
BENJAMIN

列表是被括号包含的零个或多个元素。元素可以是任何类型，包括列表本身。使用列表必须要引用，不然Lisp会自动求值。

> '(my 3 "Sons")
(MY 3 "Sons")

可以调用list创建列表。由于list是函数，它的实参会被求值。

> (list 'my (+ 2 1) "Sons")
(MY 3 "Sons")

> (list '(+ 2 1) (+ 2 1))
((+ 2 1) 3)

这是Lisp最牛逼的地方：Lisp的程序是用列表来表示的。这意味着Lisp程序可以写出Lisp代码。Lisp程序员可以（并且经常）写出能为自己写程序的程序。(这也是我最想了解的地方)

空列表：nil或者()

2.4 List Operation

cons构造列表，连接两个实参，构成一个列表。

> (cons 'a '(b c d))
(A B C D)

car,cdr取列表元素。car取列表第一个元素，cdr取列表第一个元素之后所有元素。
third取列表第三个元素。

2.5 Truth

t:真；nil：假。
函数返回将会被解释为逻辑真或者逻辑假时，则称该函数为谓词(predicate)。 CLisp中谓词通常以p结尾。
listp:是否为list

> (listp '(a b c))
T

null: 判断是否为空表
not: 逻辑取反
if: 条件式，接收三个参数：test表达式，then表达式，else表达式

> (if (listp '(a b c))
    (+ 1 2)
    (+ 5 6))
3

与quote相同，if是特殊的操作符，不能用函数来实现，因为实参在参数调用时永远会被求值。而if，只有最后两个参数中的一个被求值。

and，or：逻辑操作符。与条件式类似，接收任意数量的实参，只对能影响返回值的几个实参求值。and求值直到碰到实参为假，or求值直到碰到实参为真
这两个操作符成为宏，宏与特殊操作符一样，可以绕过一般的求值规则。第十章将介绍如何编写宏。

2.6 Functions

使用defun定义函数，通常接收三个以上的实参：一个名称，一组与列表表示的实参，以及一个或多个组成函数体的表达式。

> (defun our-third (x)
    (car (cdr (cdr x))))
OUR-THIRD
> (our-third '(a b c d))
C

defun是宏，创建一个函数并给函数命名。

Lisp 不对程序、过程、函数进行区分。函数做了所有的事情。并且Lisp尝试将执行所有的东西。
这就是为什么引用列表和符号需要加'

2.7 Recursion

递归。没什么好讲的，跟其他编程语言的一样。下面是个例子。

> (defun our-member (obj lst)
    (if (null lst)
        nil
        (if (eql (car lst) obj)
            lst
            (our-member obj (cdr lst)))))
OUR-MEMBER
> (our-member 'b '(a b c))
(B C)
> (our-member 'z '(a b c))
NIL

这一堆括号，真心难看啊。

2.8 Reading Lisp

上面刚说括号难看，这就给出方法了—靠缩进，不要看括号
一个好的编辑器会自动匹配括号，如果不能匹配，换编辑器吧。
就喜欢这种暴力的方法。

2.9 Input and Output

format 输出函数，接收两个或两个以上实参。第一个实参决定要打印到哪里，第二个参数是字符串模板，剩余参数是插入到字符串目标。
下面是一个典型的粒子：

> (format t "~A plus ~A equals ~A ~%" 2 3 (+ 2 3))
2 plus 3 equals 5.
NIL

read 输入函数，没有参数时，会读取缺省位置，通常是顶层。
需要注意的是，read是一个完成的Lisp解释器。不仅可以读入字符，它解析它所读到的东西，并返回产生出来的Lisp对象。

> (defun askem (string)
    (format t "~A" stirng)
    (read))
> (askem "How old are you?")
How old are you? 29

29

上面的函数体由两个列表组成，函数体可以包含任意数量的表达式，当函数被调用时，会依次求值，并返回最后一个的值。

2.10 Variables

let 定义局部变量，只能在let内部使用

> (let ((x 1) (y 2)) (+ x y))
3

一个更有选择性的askem函数：

> (defun ask-number ()
    (format t "Please enter a number:")
    (let ((val (read)))
        (if (numberp val)
            val
            (ask-number))))

defparameter 定义全局变量。全局变量一般形式是*name*，即“星-命-星”结构

> (defparamter *glob* 99)
*GLOB*

也可以用defconstant定义一个全局常量：

> (defconstant limit (+ *glob* 1))

判断是否为全局变量或者常量，使用boundp函数

> (boundp '*glob*)

2.11 Assignment

setf 赋值操作符。可以用来给全局或者局部变量赋值。

> (setf *glob* 98)
98
> (let ((n 10)) (setf n 2) n)
2

如果setf的第一个实参是符号(symbol)，且符号不是某个局部变量的名字，则setf将这个符号设置为全局变量。即通过赋值，可以隐式的创建全局变量。
setf的第一个实参还可以是变量或者表达式，这种情况下，第二个实参将被插入到第一个实参所引用的位置。

> (setf (car '(a b c)) 'n)
N

还可以传入偶数个实参，依次赋值。

2.12 Functional Programming

函数式编程是撰写利用返回值而工作的程序。它是Lisp的主流范式。不大明白，不就是函数么？
函数式编程是为了返回值，而不是修改。例如，remove接收一个对象和一个列表，返回不包含这个对象的新列表，原列表并未被修改。

如果移除一个元素，Lisp里经常这么做：

> (setf x (remove 'a x))

但不建议这么做。
函数式编程本质上意味着避免使用如setf的函数。
函数式编程最重要的一个优点是：允许交互测试(interactive testing)。

2.13 Iteration

递归。

> (defun show-squares (start end)
    (do ((i start (+ i 1))) ((> i end) 'done)
        (format t "~A ~A ~%" i (* i i))))

do宏 是Clisp中最基本的迭代操作符。
上式的递归版是：

> (defun show-squares (i end)
        (if (> i end) 'done 
            (progn (format t "~A ~A ~%" i (* i i))
                (show-squares (+ i 1) end))))

progn，接收任意数量的表达式，依次求值，并返回藏族一个表达式的值。
dolist 遍历列表

2.14 Functions as Objects

function 特殊操作符，把一个函数的名字传递给function，它会返回相关联的对象。

> (function +)
#<Complied-Function + 7BA4E>

有点像C里面的取地址符&，获取指向函数的指针
如同'作为quote的缩写，可以用#'作为function的缩写，这个缩写称之为升引号。
apply 接收一个函数和实参列表，并返回把传入函数应用在实参列表的结果。

> (apply #'+ '(1 2 3))
6

funcall 做同样的事情，但不需要将实参包装成列表

> (funcall #'+ 1 2 3)
6

lambda 两种方式：

((x) (+ x 100))

或者

(lambda (x) (+ x 100))

实验表明，最好还是加上lambda，不然会出错

2.15 Types

Lisp里只有数值才有类型，变量没有类型，变量可以存放任何类型的对象。
数值类型，依普遍性的增序排列依次是：fixnum,integer,rational,real,number,atom,t，类型t是所有类型的基类。
typep 接收一个对象和一个类型，然后判断对象是否属于该类型。

> (typep 27 'integer)
T

2.16 Looking Forward

本章讨论Lisp的语法，可以看到具有以下特点：

  • 语法单一。
  • 语法基于列表，列表是一种Lisp对象
  • 函数也是Lisp对象，函数也能用列表表示
  • Lisp本身就是Lisp程序，自定义的函数与内置的Lisp函数没有任何区别。
